# MERN POSTAPP
> This App is built on Mern stack with redux toolkit and jwt integration <br>
> User can register, login , addpost , edit and delete <br>
> For global states Redux Toolkit is used <br>
> JWT is used so after 24Hours user will be automatically logut <br>
> Fully Responsive

## Technologies
> React <br>
> Nodejs <br>
> Express js <br>
> MongoDB <br>
> JWT <br>
> Redux Toolkit <br>
